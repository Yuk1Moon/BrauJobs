# Changes Hair Landing
Experiencia premium de una sola página para Changes Hair.

## Ver la página localmente
```bash
./serve.sh
```
Luego abre `http://localhost:8000` en tu navegador para explorar el sitio.

## Prueba rápida de estructura
Ejecuta el smoke test para validar que la landing conserva las secciones clave y los elementos solicitados:
```bash
python smoke_test.py
```
